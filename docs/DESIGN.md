## 2. жЁЎеқ—и®ҫи®Ў

### 2.1 Agent жЁЎеқ—

**иҒҢиҙЈ**: еҚҸи°ғеҗ„ Skill е®ҢжҲҗе·ҘдҪңжөҒ

**зұ»и®ҫи®Ў**:
```python
class ArXivBlogAgent:
    - config: AgentConfig        # й…ҚзҪ®
    - status: AgentStatus        # зҠ¶жҖҒ
    - skills: Dict[str, Skill]   # жҠҖиғҪжіЁеҶҢиЎЁ

    + __init__(config_path, dry_run)
    + run() -> AgentResult
    + get_status() -> Dict
    + close()
```

**зҠ¶жҖҒжңә**:
```
IDLE -> INITIALIZING -> SCRAPING -> GENERATING -> POSTING -> COMPLETED
  |         |            |            |            |
  в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙ-> FAILED
```

## 5. жҺҘеҸЈи®ҫи®Ў

### 5.1 е‘Ҫд»ӨиЎҢжҺҘеҸЈ

```bash
# еҹәжң¬иҝҗиЎҢ
python src/main.py

# е№ІиҝҗиЎҢжЁЎејҸ
python src/main.py --dry-run

# иҮӘе®ҡд№үй…ҚзҪ®
python src/main.py -c config.internal.yaml

# иҮӘе®ҡд№үжҗңзҙў
python src/main.py -q "transformer attention" -m 5

# иҜҰз»Ҷж—Ҙеҝ—
python src/main.py -v
```

- **еҚ•е…ғжөӢиҜ•**: жҜҸдёӘ Skill зҡ„зӢ¬з«ӢжөӢиҜ•
