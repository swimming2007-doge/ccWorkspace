## [0.1.0] - 2026-03-06

### 新增

#### 核心功能

- **F004 动作检测核心算法** (手表端)
  - 传感器数据采集服务 (加速度计、陀螺仪、心率、GPS)
  - 数据缓冲队列 (滑动窗口 5秒@50Hz)
  - 特征提取算法 (时域+频域)
  - 动作分类器 (决策树)
  - 骑行状态机 (IDLE/RIDING/PAUSED/ENDING)

- **F005 手表端骑行界面** (手表端)
  - 数据展示组件 (DataCard, HeartRateRing, SpeedDisplay)
  - 骑行中页面 (实时速度/距离/时间/心率)
  - 骑行总结页面 (统计数据、心率区间分布)

- **F006 手机-手表BLE通信** (手机+手表)
  - BLE通信协议定义 (UUID、数据包格式)
  - BLE协议工具类 (分包/重组)
  - 手表端BLE服务 (广播、连接、数据发送)
  - 手机端BLE客户端 (扫描、连接、数据接收)
  - 断线重连机制

- **F007 GPS数据采集** (手机+手表)
  - 手机端GPS服务 (高精度定位)
  - 手表端GPS服务 (低功耗定位)
  - 卡尔曼滤波轨迹平滑
  - 道格拉斯-普克算法简化
  - GPS信号丢失处理 (位置外推)
  - 手机+手表GPS数据融合

- **F008 心率数据采集** (手表)
  - 心率传感器数据采集 (1Hz)
  - 心率区间计算 (Z1-Z5)
  - 心率异常检测 (过高/过低/不规律)
  - 心率统计数据 (平均/最大/最小/区间分布)

- **F009 手机端实时数据看板** (手机)
  - 骑行数据管理服务
  - 数据展示组件 (DataCard, HeartRateRing, SpeedDisplay)
  - 实时看板页面 (速度/距离/时间/心率/海拔)
  - 心率区间可视化 (环形图 + 分布条)
  - 地图轨迹实时绘制
  - 支持横竖屏切换

- **F010 骑行记录存储** (手机)
  - 数据库服务
  - 骑行记录存储服务
  - 数据导出服务 (GPX/TCX/JSON)
  - 轨迹点按月分表优化

- **F013 公路导航模式** (手机)
  - 导航服务 (路线规划、状态管理、语音提示)
  - 导航界面组件 (转向提示卡片、导航信息面板)
  - 偏离路线检测
  - 导航控制按钮

- **F014 云端同步服务** (手机+服务端)
  - 手机端同步服务 (同步状态管理、进度追踪)
  - 断点续传、增量同步
  - 自动同步和批量同步
  - 服务端OBS存储服务
  - 华为云OBS SDK集成
  - 分片上传支持

- **F015 用户认证服务** (手机+服务端)
  - 手机端认证服务 (华为账号OAuth登录)
  - JWT Token管理和刷新
  - 登录状态持久化
  - 游客模式登录

#### 扩展功能

- **F016 探险导航模式** (手机)
  - 探险路网数据服务
  - 支持土路、村道、山地车道、小径导航
  - 混合路线规划算法
  - 路况预警提示

- **F017 冒险指数计算** (手机)
  - 6因素评估模型 (路面、坡度、交通、信号、补给、天气)
  - 高程数据服务
  - 天气数据服务
  - 信号覆盖服务
  - 实时风险等级评估
  - 安全建议生成

- **F018 离线地图** (手机)
  - 离线地图区域管理
  - 瓦片下载服务
  - 存储空间管理
  - 下载任务管理 (开始/暂停/继续)
  - LRU清理策略

- **F019 统计分析模块** (手机)
  - 日/周/月/年统计
  - 趋势分析
  - 心率区间分布
  - 热量消耗计算
  - 个人记录

- **F020 Web端数据看板** (Web)
  - Vue 3 + TypeScript + Vite 项目
  - 数据看板页面
  - 骑行列表和详情页面
  - 统计分析页面

- **F021 微信分享功能** (手机)
  - 微信好友/群分享
  - 朋友圈分享
  - 4种分享模板 (简约/运动/探险/数据)
  - 分享内容定制

- **F022 Strava同步** (手机+服务端)
  - OAuth 2.0 认证
  - GPX 文件生成和上传
  - Token 管理和刷新
  - Webhook 事件处理

- **F023 用户设置模块** (手机)
  - 骑行偏好设置
  - 动作检测设置
  - 同步设置
  - 隐私设置

- **F024 性能优化与NFR验证** (全平台)
  - 手机端性能监控服务
  - 手表端性能监控服务
  - 服务端性能监控服务
  - Web端性能监控服务
  - NFR指标验证框架

#### 基础设施

- **F001 项目骨架与配置**
  - 项目目录结构
  - 共享类型定义
  - 共享工具函数
  - 共享常量

- **F002 数据库Schema设计**
  - 手机端SQLite表结构
  - 服务端Prisma Schema

- **F003 服务端API骨架**
  - Express应用入口
  - 路由骨架
  - JWT认证中间件

- **F003-Docker Docker容器化部署**
  - 多阶段Dockerfile
  - docker-compose配置
  - 数据卷持久化
  - 健康检查端点

### 技术架构

- 采用轻量化单体架构
- 手机端: HarmonyOS + ArkTS + SQLite
- 手表端: Watch OS + ArkTS
- 服务端: Node.js + Express + SQLite + 华为云OBS
- Web端: Vue 3 + TypeScript + Vite
